Hakkımda

Ben Hacettepe Üniversitesi Geomatik Mühendisliği 4. sınıf öğrencisiyim. Kariyerime maden sektöründe devam etmeyi planlıyorum. İlk olarak, Basarsoft'ta Web Yazılım Departmanı'nda .NET Web API kullanarak harita tabanlı bir web sitesi geliştirdim. Bu deneyimim, yazılım geliştirme süreçlerini ve harita teknolojilerini derinlemesine anlamamı sağladı. Daha sonra, Öksüt Madencilik/Centarra Gold'da staj yaptım. Bu süreçte, maden sektöründe uzaktan algılama tekniklerini kullanarak çalıştım. Bu deneyim, sektörün dinamiklerini anlamama ve teknik becerilerimi geliştirmeme yardımcı oldu. Gelecekte, maden sektöründeki gelişmeleri takip ederek bu alanda uzmanlaşmayı hedefliyorum.
